# Budget Tracker (Python)

## Overview
A simple Python project to track personal expenses from a CSV file.  
Shows total spending, category summaries, and bar chart visualization.

## Features
- Calculate total spending
- Summarize spending by category
- Visualize spending with bar charts
- Reads from CSV for easy data management

## How to Run
1. Clone repo
2. Install Python 3.8+
3. Install dependencies: `pip install pandas matplotlib`
4. Run: `python budget_tracker.py`
5. Ensure `expenses.csv` is in the same folder
